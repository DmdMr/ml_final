# Credit Scoring (UCI Credit Card)

Проект предсказывает вероятность дефолта клиента по кредитной карте в следующем месяце и рассчитывает кредитный рейтинг/категорию риска.

## Что было не так
В исходном ноутбуке переменная `y` использовалась до объявления в `train_test_split`, из-за чего пайплайн падал с `NameError`.

## Исправление
Добавлен рабочий скрипт `main.py`, который:
- загружает и подготавливает данные;
- выполняет one-hot encoding категориальных признаков;
- добавляет engineered-признаки (`avg_bill_amt`, `avg_pay_amt`, `pay_ratio`);
- обучает `RandomForestClassifier`;
- выводит ROC-AUC, отчёт классификации и confusion matrix;
- рассчитывает `PD`, `Credit_Score` и `Risk_Category`.

## Запуск
```bash
python main.py
```
